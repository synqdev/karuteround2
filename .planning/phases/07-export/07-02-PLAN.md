  <files>src/lib/karute/formatKaruteText.ts, src/app/api/karute/[id]/export/text/route.ts</files>
  <action>
Create `src/lib/karute/formatKaruteText.ts` -- a pure function with no external dependencies:

1. Export `formatKaruteAsText(karute: KaruteWithRelations): string`.
2. Import the KaruteWithRelations type from wherever Phase 4 established it (`@/lib/supabase/karute` or `@/types/karute`).
3. Format as structured plain text with clear section separators:
   - Line 1: "カルテ"
   - "顧客: {customer name}" (or "---" if no customer)
   - "日付: {created_at formatted as ja-JP locale date}"
   - Blank line
   - "===== AI サマリー ====="
   - Summary text (or empty string if null)
   - Blank line
   - "===== エントリー ====="
   - For each entry: blank line, "[{category}]", content text, and if source_quote exists: `  引用: "{source_quote}"`
   - Blank line
   - "===== トランスクリプト ====="
   - Transcript text (or empty string if null)
4. Join all lines with `\n` and return.

Create `src/app/api/karute/[id]/export/text/route.ts`:

1. Add `export const runtime = 'nodejs'` at top.
2. Import `createClient` from `@/lib/supabase/server`, `getKaruteRecord` from `@/lib/supabase/karute`, `formatKaruteAsText` from `@/lib/karute/formatKaruteText`.
3. Export async `GET` handler:
   a. Extract `id` from route params.
   b. Auth check: create Supabase client, get user, return 401 if no user.
   c. Fetch karute: `getKaruteRecord(id)`, return 404 if not found.
   d. Format: `const text = formatKaruteAsText(karute)`.
   e. Build filename: `karute-{customerName}-{date}.txt`.
   f. Return `new Response(text, { headers: { 'Content-Type': 'text/plain; charset=utf-8', 'Content-Disposition': 'attachment; filename="..."' } })`.
