<task type="auto">
  <name>Task 3: Whisper transcription API route</name>
  <files>src/app/api/ai/transcribe/route.ts</files>
  <action>
Create `src/app/api/ai/transcribe/route.ts`:
- Export `maxDuration = 60` at the top (Vercel Pro timeout)
- Export async `POST` handler that:
  1. Reads FormData from request: `audio` (File) and `locale` (string, default 'ja')
  2. Validates `audio` exists — return 400 `{ error: 'No audio provided' }` if missing
  3. Converts File to Buffer: `Buffer.from(await audioFile.arrayBuffer())`
  4. Determines MIME type from `audioFile.type` — default to `audio/webm` if empty
  5. Determines file extension: `audio.mp4` if MIME contains 'mp4', else `audio.webm`
  6. Calls `openai.audio.transcriptions.create()` with:
     - `file: await toFile(buffer, extension, { type: mimeType })` (import `toFile` from `openai`)
     - `model: 'gpt-4o-mini-transcribe'` (per research recommendation)
     - `language: locale === 'ja' ? 'ja' : 'en'`
     - `response_format: 'text'`
  7. Returns `NextResponse.json({ transcript: transcription })`
  8. Wraps in try/catch — on error, return 500 `{ error: 'Transcription failed' }`

Import `openai` from `@/lib/openai`. Import `toFile` from `openai`.
Audio buffer goes out of scope after the response — never persisted (AI-05 compliance).
  </action>
